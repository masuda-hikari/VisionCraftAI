## 技術的詳細

### 採用技術
- **テンプレートエンジン**: Jinja2（FastAPI統合）
- **静的ファイル配信**: FastAPI StaticFiles
- **フロントエンド**: HTML5/CSS3/Vanilla JavaScript
- **スタイリング**: カスタムCSS（グラデーション、レスポンシブ）

### ランディングページ機能
1. **ヒーローセクション**: サービス概要・CTA
2. **特徴セクション**: 6つの主要機能紹介
3. **料金セクション**: 4プラン比較表示
4. **デモセクション**: 画像生成体験UI
5. **フッター**: 製品・リソース・サポートリンク

### デモ機能
- プロンプト入力
- 解像度選択（512/1024/2048px）
- スタイル選択（フォトリアリスティック/アーティスティック等）
- APIキー自動作成・保存
- 生成結果表示
